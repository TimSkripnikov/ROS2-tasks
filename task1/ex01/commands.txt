===
Название: top
Описание: Утилита для мониторинга процессов в реальном времени. Показывает загрузку CPU, RAM, процессы.
Синтаксис: top [опции]
Пример: top
Примечание: Нажав q — выход, k — завершение процесса, h — помощь
===

===

Описание: Улучшенный интерактивный монитор процессов для Unix-систем.
Синтаксис: htop [опции]
Пример: htop

===

===
Название: man
Описание: Просмотр документации для команды.
Синтаксис: man <команда>
Пример: man grep
Пример: man ls
Примечание: Навигация стрелками, выход по клавише q
===

===
Название: grep
Описание: Поиск строк, содержащих указанный шаблон.
Синтаксис: grep [опции] "шаблон" файл
Пример: grep "error" logfile.log
Примечание: Опция -i позволяет игнорировать регистр, -r ищет рекурсивно в папке
===

===
Название: tar
Описание: Архиватор для упаковки и распаковки файлов.
Синтаксис: tar [опции] [архив] [файлы]
Пример: tar -czvf archive.tar.gz folder/
Примечание: -c создать, -x извлечь, -v показать процесс, -f указать имя файла
===

===
Название: df -h
Описание: Просмотр свободного и занятого места на дисках.
Синтаксис: df [опции]
Пример: df -h
Примечание: -h отображает размеры в удобном виде (MB, GB)
===

===
Название: du
Описание: Показывает размер файлов и папок.
Синтаксис: du [опции] путь
Пример: du -sh *
Пример: du -sh ~/Downloads
Примечание: -s суммарно, -h в удобном формате
===

===
Название: wget
Описание: Утилита для скачивания файлов из интернета.
Синтаксис: wget [опции] URL
Пример: wget https://example.com/file.zip
Примечание: Можно использовать -O для указания имени файла
===

===
Название: alias
Описание: Создание псевдонимов для команд.
Синтаксис: alias имя='команда'
Пример: alias ll='ls -lh'
Пример: alias gs='git status'
Примечание: Ускоряет работу, алиасы можно сохранять в ~/.bashrc
===

===
Название: unalias
Описание: Удаление ранее созданного alias.
Синтаксис: unalias имя
Пример: unalias ll
Примечание: После этого команда будет выполняться в оригинальном виде
===

===
Название: !!
Описание: Повторение последней выполненной команды.
Синтаксис: !!
Пример: !!
Примечание: Прикольно, хотя стрелками лучше
===

===
Название: history
Описание: Показ списка последних введённых команд.
Синтаксис: history
Пример: history
Примечание: Чтобы долго не листать стрелками
===

===
Название: !45
Описание: Выполнение команды по её номеру из истории.
Синтаксис: !номер
Пример: !45
Примечание: Номер можно взять из списка, полученного командой history
===
